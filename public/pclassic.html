The basics of JavaScript!

JS is a simple language. There are just
6 types* a variable can have:

  1. Numbers:   3.14159...
  2. Booleans:   true/false
  3. Strings:   "Hello World!"
  4. Arrays:   ["foo", "bar"]
  5. Hashes:   {foo: "bar", age: 42}
  6. Functions:    function(a, b){return a+b;}
*Okay, 7 counting regular expressions, but let's ignore them...

// Let's put it all together.
var x = 12;
var y = 19;
var arr = [x, y];
var hash = {vals : arr};
function func(a,b){ return a + b};
var ans = func(hash.vals[0], hash.vals[1]);
console.log(ans);
Recently, people have started to use JavaScript outside the browser using Node.js.
This means you can use JavaScript anywhere you could use a language like Java or Python — full-stack JavaScript.
Full-stack JavaScript is cool because you only need to use one language to write web applications. Frameworks like Express.js and Meteor make this easy.

For example, check out pclassic.meteor.com, a simple chat app written only in JavaScript!
